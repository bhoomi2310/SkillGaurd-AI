# SkillBridge AI - Frontend

## Quick Start

1. Install dependencies: `npm install`
2. Create `.env` file (optional, defaults to `/api`)
3. Run: `npm run dev`

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build

## Project Structure

- `src/pages/` - Page components
- `src/components/` - Reusable components
- `src/services/` - API service layer
- `src/context/` - React context providers

## Features

- Student dashboard with skill charts
- Task marketplace with filtering
- Submission form (GitHub/file upload)
- Provider dashboard
- Recruiter search and student profiles
